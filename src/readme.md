# IgniteLab

This repo contains the Ignite Lab demo. It has two folders, **Start**, with the initial setting for the lab, and **Final**, with the final project

## Getting Started

Go to the folder **Start**, open README.md in a markdown viewer and follow the steps to complete the lab

### Installing

Clone this repo. In Visual Studio, open _ContosoLab.sln_ to open the existing WPF project. You should have a UI shell, that will be filled in the lab
